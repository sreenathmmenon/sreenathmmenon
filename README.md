## 🚀 What I'm building

**[llmswap](https://llmswap.org)** · `41K+ installs`
Provider-agnostic LLM SDK + CLI for 11 providers (OpenAI, Anthropic, Gemini, Groq, Watsonx, Ollama, xAI, Sarvam…) with zero vendor lock-in. Workspace memory, universal tool-calling, an MCP client, and cost visibility.
[PyPI](https://pypi.org/project/llmswap/) · [Docs](https://llmswap.org) · [GitHub](https://github.com/sreenathmmenon/llmswap)

**[Krelvan](https://krelvan.com)** · `self-hostable, event-sourced`
Describe an outcome in plain English and get a fully owned, running multi-agent system. A signed, append-only event ledger *is* the runtime, so what you see is exactly what executed; agents crash and resume with no action repeated. Prompt-injected instructions can only shrink an agent's powers, never expand them.
[krelvan.com](https://krelvan.com) · [GitHub](https://github.com/sreenathmmenon/krelvan)
